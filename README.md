# Calculator (Python)

## 📌 Description

This is a simple Calculator application developed using Python. It performs basic arithmetic operations such as addition, subtraction, multiplication, and division. This project was created to practice Python programming fundamentals.

## ✨ Features

* Addition
* Subtraction
* Multiplication
* Division
* User-friendly menu
* Input validation for basic operations

## 🛠️ Technologies Used

* Python 3

## ▶️ How to Run

1. Make sure Python is installed on your computer.
2. Download or clone this repository.
3. Open the project in VS Code or any Python IDE.
4. Run the Python file.

## 📷 Output

The program displays a menu, accepts user input, performs the selected operation, and shows the result.
